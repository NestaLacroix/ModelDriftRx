Morning — starting work:
  $ make run-api                 # API running on :8000
  $ make run-dashboard           # dashboard on :8501 (separate terminal)

Writing code:
  $ make format                  # auto-format before you forget
  $ make test-fast               # quick feedback loop (skips slow SHAP tests)

Before pushing:
  $ make check                   # runs lint + format + type-check
  $ make test-cov                # full tests + see coverage gaps

Demoing the project:
  $ make demo                    # one command, full self-healing loop

When things get messy:
  $ make clean                   # nuke all caches and start fresh
